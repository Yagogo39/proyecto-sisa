# SIS-A - Sistema de Papelería "Don Max"

Este repositorio contiene el sistema integral para la papelería, dividido en:

- **/frontend**: React + Vite + Tailwind CSS v4.
- **/backend**: Node.js + Express + SQLite.

## Instalación para el equipo

1. Clonar el repo: `git clone https://github.com/Yagogo39/proyecto-sisa.git`
2. Configurar Frontend: 
   - `cd frontend`
   - `npm install`
3. Configurar Backend:
   - `cd ../backend`
   - `npm install`

## Notas
- No subir el archivo `database.db` (está en .gitignore).
- Usar el puerto 3000 para el servidor y 5173 para el cliente.
